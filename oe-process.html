逻辑流总览

分板站位 → 耦合目检 → 自动耦合测试 → 耦合外观检测 → 耦合烘烤 → TXFA组装 → 端面清洁 → MPD测试 → OE测试 → TCT快速温变 → 模块BI → 热岛立柜 → 端面检测清洁 → OE/TRX测试 → OE出货检查

OE光引擎区是从「耦合后」到「出货前」的完整工序闭环。分为测试验证区（工序性能验证）和可靠性试验区（极限环境筛选）两大板块。每个工位的测试结果作为"过站"条件，只有当前工位合格才放行至下一工位。

5. 智能料箱处理系统

在OE光引擎这类精密制造区，料箱的处理方式直接决定了产线的效率和良率。现代光模块产线已普遍采用全流程自动化+智能化调度的模式来处理料箱，其核心是减少人工干预，实现物料搬运和上下料的无人化。

5.1 自动化上下料：从"人找料"到"料找人"

光模块的上下料环节已高度自动化。设备集成自动上下料模块，利用多层料仓与多PCS料盘结构，实现一次上料后的批量、持续生产。料箱通过输送线、提升机等设备，被精准分配到各个工位。

要素说明
料仓结构多层抽屉式料仓，每层承载标准料盘（Tray），单次上料可支持 2~8 小时连续生产
料盘规格多PCS料盘（JEDEC标准），每盘承载 20~100 颗光引擎，兼容不同封装尺寸
取放精度±50μm 料盘定位，视觉引导吸嘴/夹爪取放，兼容不同器件外形
换料节拍空满料盘交换 <15s，不停机换料
缓存策略工位旁设置 3~5 层线边缓存仓，缓冲上下游节拍差异

5.2 Z100 · 基础上下料机器人

Z100 是产线的基础上下料执行单元，核心能力为料盘→设备→料盘的标准取放循环。

能力参数
核心功能料箱取料 → 设备上料 → 成品下料 → 料箱归位
负载≤3kg（单颗光引擎 ~50g，整盘 ~5kg 可分批）
重复定位±0.1mm
视觉引导2D相机料盘定位 + 凹槽识别，兼容多规格Tray
对接设备耦合台、测试站、烘烤炉——标准料盘接口
安全机制力矩传感器防碰撞 · 光栅安全区域 · ESD接地 <100V
典型节拍取料 3s + 上料 5s + 下料 3s + 归位 4s = 单循环 ≤15s
部署密度每 3~5 台设备配 1 台 Z100，负责该设备群的上下料

5.3 Z700 · 具身智能上下料+搬运一体化机器人

Z700 是 OE 区的核心物流执行单元，在 Z100 的上下料能力基础上，增加了工位间搬运与智能调度能力。双臂设计使其可同时处理「取+放」操作，大幅提升节拍。

能力维度Z100（基础型）Z700（升级型）
核心功能固定工位上下料上下料 + 工位间搬运 + 跨区传递
手臂配置单臂双臂（可同时取+放）
移动能力固定底座轮式自主导航（AMR底盘）
负载≤3kg≤5kg × 2臂
视觉系统2D 料盘定位3D 结构光 + RGB-D，料箱/料盘/散料全场景识别
料箱对接固定料仓位多层缓存仓 + 线边料架 + AGV对接位，自主导航至任意料箱点
调度对接PLC I/O 信号MES HTTP/WebSocket 实时任务接收 + 状态反馈
典型应用单台耦合测试站上下料14台耦合台群的物料配送 + 耦合→外观→烘烤工位间传递
覆盖范围1台设备整个OE区（I125~I139），15个工位全覆盖

5.4 智能物流搬运：AGV/AMR

载体功能覆盖
AGV A2OE区边界巡逻 + 料箱转运，按MES指令从线边仓取料送至工位群入口OE区右边界
AMR（规划中）自主避障、动态路径规划，替代固定磁轨AGV，支持产线布局灵活调整全车间
输送线连接烘烤炉→测试站→老化柜的固定料箱通道，与机器人协同形成混合物流网络设备间固定链路

5.5 多层料仓与线边缓存

缓存层位置容量用途
L1 入口缓存OE区入口（分板站旁）8层 × 20盘接收COC区来料，等待分板
L2 耦合缓存耦合台群旁5层 × 10盘分板后待耦合器件，缓冲14台耦合台并行需求

5.6 MES 智能调度与追溯

所有环节由统一的制造执行系统（MES）指挥。MES根据生产计划和实时产线状态，智能调度AGV、Z100/Z700机器人、输送线等执行单元。

系统能力说明
任务分发MES → Z700/WiFi：取料任务 {料箱ID, 起点缓存层, 终点工位, 优先级}
料箱追踪每个料箱贴 RFID 标签 + 二维码，出入缓存仓自动扫码，全生命周期可追溯
物料追溯单颗器件级追溯：从COC基板ID → 耦合参数 → 测试数据 → 出货批次，全链路绑定
动态调度设备故障/节拍波动时自动重新分配料箱路线，Z700 接收实时改派指令
状态看板产线数字孪生大屏：实时显示每台Z700位置、每个缓存仓料位、每台设备状态
异常处理料箱卡料/空仓/定位失败 → MES 告警 → 人工介入或自动重试（最多3次）
OE光引擎区的料箱处理，是一个由Z100基础上下料 + Z700具身智能搬运 + AGV/AMR物流 + 多层缓存仓 + MES调度系统 协同构成的完整闭环，旨在实现高效、精准、可追溯的物料流转。Z100 确保单站料不中断，Z700 打通工位间壁垒，MES 统一调度全局最优。
