1. 定位与范围

1.1 OE 定义

项说明
全称Optical Engine / 光引擎
物料组成LD芯片+透镜+隔离器+光纤阵列+MPD+热沉
结合方式UV固化耦合 + 激光焊接 + 金丝键合
阶段产出耦合封装完成的光引擎组件(OSA)
产线位置储翰 OE 车间

1.2 数通 OE 特点

对比项数通电信
封装形式BOX/蝶形TO-CAN/蝶形
耦合精度±0.5μm±1μm
通道数单通道/4通道单通道
测试项IL/RL/PDL/光谱/眼图IL/RL/SMSR
自动化14台自动耦合 · AOI检测半自动耦合
数通OE耦合是光模块核心工艺，精度要求高(±0.5μm)，适合Z700轮式机器人做治具传递与自动上下料。14台耦合台并行是产能瓶颈，需重点优化节拍。

逻辑流总览

分板站位 → 耦合目检 → 自动耦合测试 → 耦合外观检测 → 耦合烘烤 → TXFA组装 → 端面清洁 → MPD测试 → OE测试 → TCT快速温变 → 模块BI → 热岛立柜 → 端面检测清洁 → OE/TRX测试 → OE出货检查

OE光引擎区是从「耦合后」到「出货前」的完整工序闭环。分为测试验证区（工序性能验证）和可靠性试验区（极限环境筛选）两大板块。每个工位的测试结果作为"过站"条件，只有当前工位合格才放行至下一工位。
