四、Sys2 HTTP API (4090:50052)

GET /health

curl http://10.23.24.0:50052/health
# → {"status":"ok","gpu":"RTX 4090","vram_gb":25.3,"models":["vtla"],"ready":true}

POST /predict

curl -X POST http://10.23.24.0:50052/predict \
  -H 'Content-Type: application/json' \
  -d '{"engine":"vtla","observation":{"task":"insert optical module"}}'
# → {"action":[0.12,-0.03,0.45,...],"model_used":"vtla","latency_ms":284}

五、gRPC 断连回退

# xspace implement · 三引擎自动降级
VTLA unreachable → 自动回退 ACT (零动作安全输出)
GR00T unreachable → 自动回退 ACT
# 连接恢复 → 自动切回远程引擎
