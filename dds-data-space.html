五、Web训练对接 (DDS → 训练)

数据采集 → LeRobotDataset 转换

# xspace 全局数据节点 → rosbag → 训练数据
ros2 bag record -o train_data /dds/realsense /dds/joint_state /dds/act_action /dds/vtla_force

# web 训练端拉取 (从ECS)
wget http://datadrive.world/data/train_data.tar.gz

# 导入训练管线
python unified_train.py --model vtla --dataset train_data/
python unified_train.py --model groot --dataset train_data/

训练权重回传 (训练 → DDS)

# web → ECS
scp model.pt root@39.102.211.79:/www/wwwroot/datadrive.world/models/

# ECS → Orin (小芳部署)
wget http://datadrive.world/models/vtla_trained.pt

# DDS Skill 7 广播权重更新
ros2 topic pub /dds/weights std_msgs/String "data: 'vtla_trained.pt'"
