四、训练命令参考

4090 统一训练管线

# VLA-T 训练 (多轮)
python unified_train.py --model vtla --rounds 5

# GR00T 微调 (需要MetaWorld数据集)  
python unified_train.py --model groot --dataset /path/to/metaworld

# 检查数据闭环连通性
python unified_train.py --ping

ECS 数据中转

# 4060上传训练数据到ECS
scp training_data.tar.gz root@39.102.211.79:/www/wwwroot/datadrive.world/data/

# 4090从ECS拉取数据
wget http://datadrive.world/data/training_data.tar.gz

# 4090训练完成 → 权重回传ECS
scp model.pt root@39.102.211.79:/www/wwwroot/datadrive.world/models/
