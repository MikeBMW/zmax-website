🗂️ DDS Skill 主数据体系

model_skill(多模态训练) + action_skill(光模块动作) · 全主数据 · 2026-07-14

🧠 model_skill · 模型技能

多模态数据 → 模型训练输入

M01 · sensor_rgb

DDS Topic/dds/model/rgb
数据类型uint8 (B,3,H,W)
来源RealSense D435 → Orin
训练模型Sys11/Sys21/Sys22
质量Key帧率≥25fps, 1920×1080

M02 · sensor_depth

DDS Topic/dds/model/depth
数据类型float32 (B,1,H,W)
来源RealSense D435 深度流
训练模型Sys21 VTLA(3D感知)
质量Key0.1-10m有效范围

M03 · sensor_force_6d

DDS Topic/dds/model/force
数据类型float32[6] (Fx,Fy,Fz,Tx,Ty,Tz)
来源六维力传感器
训练模型Sys21 VTLA(力控)
质量Key±0.1N精度, 1kHz

M04 · sensor_tactile

DDS Topic/dds/model/tactile
数据类型float32[16] (指尖触觉阵列)
来源指尖触觉传感器
训练模型Sys21 VTLA(触觉编码器)
质量Key16通道·16bit·1kHz

M05 · joint_state

DDS Topic/dds/model/joint
数据类型float32[7] + float32[7] + float32[7]
来源6轴机械臂编码器
训练模型Sys1/Sys11
质量Key±0.01°, 1kHz

M06 · task_language

DDS Topic/dds/model/language
数据类型string (任务指令)
来源DeepSeek V4 Pro / 人工
训练模型全部VLA模型
质量Key长度≤256 tokens

M07 · expert_demo

DDS Topic/dds/model/demo
数据类型LeRobotDataset shard
来源Genesis仿真 / 人工示教
训练模型全部6模型
质量Key帧数≥100, 动作范围合法

M08 · sim_data

DDS Topic/dds/model/sim
数据类型Genesis仿真输出
来源Genesiw-World 物理引擎
训练模型全部6模型
质量Key物理参数有效, 无穿透

⚡ action_skill · 动作技能

光模块工厂 · 围绕插拔动作

A01 · pick_module

DDS Topic/dds/action/pick
工序从料盘取光模块
型号适配800G OSFP / 400G QSFP-DD / 100G SFP
引擎Sys1 ACT (1.2ms)
力控阈值夹持力 3-8N

A02 · scan_barcode

DDS Topic/dds/action/scan
工序扫码读取SN
型号适配所有型号
引擎Orin本地视觉
校验SN匹配MES系统

A03 · pre_insert

DDS Topic/dds/action/pre_insert
工序定位到插口上方
型号适配800G(大)/400G(中)/100G(小) 不同预插位
引擎Sys11 SmolVLA (视觉引导)
精度XY ±0.5mm

A04 · lissajous_insert ⭐

DDS Topic/dds/action/lissajous
工序利萨如力控搜索插入 (核心)
型号适配QSFP-DD/OSFP不同金手指宽度
引擎Sys21 VTLA (力控284ms)
力控参数Z 6N, 振幅6mm/3Hz+4mm/2Hz, 搜索±10mm

A05 · second_insert

DDS Topic/dds/action/second_insert
工序二次插入确认 (end_pose[0]>0.75)
引擎Sys21 VTLA
回退失败→开爪重试→连续2次→报错

A06 · extract_module

DDS Topic/dds/action/extract
工序Z+12cm 直线拔出
型号适配所有型号统一流程
引擎固定轨迹 → joint24 → joint14

A07 · place_ok

DDS Topic/dds/action/place_ok
工序放回合格品托盘
目标位姿manual_pose_25

A08 · place_ng

DDS Topic/dds/action/place_ng
工序放到不合格品区
目标位姿manual_pose_26

A09 · aoi_inspect

DDS Topic/dds/action/aoi
工序6工位AOI检测
引擎4060:10082 capture_detect
判断any NG → 标记aoi_fail_flag

A10 · test_wait

DDS Topic/dds/action/test
工序等待测试结果 (PASS/FAIL)
超时30s

A11 · model_change

DDS Topic/dds/action/change
工序换型适配 (切换光模块型号)
参数新型号→更新预插位/力控参数/夹持力

A12 · init_cycle

DDS Topic/dds/action/init
工序初始化循环 (计数器清零)
计数器pick_place/place/ng/grasp_fail/aoi_detect

DDS API

from dds_skill import gds

# model_skill (多模态训练数据)
gds.publish("sensor_force_6d", [1.2, 0.3, 5.0, 0, 0, 0])  # M03
gds.get("joint_pos_7d")                                      # M05

# action_skill (光模块动作)
gds.publish("pick_module", {"model":"QSFP-DD", "force":5.0})  # A01
gds.publish("lissajous_insert", {"z_force":6.0, "amplitude":6.0})  # A04 ⭐
gds.get("aoi_result")                                         # A09

# 全部主数据 → Web看板
gds.to_json("all")  # 20 Topics → data-quality.html 实时监控
